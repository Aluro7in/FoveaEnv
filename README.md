# FoveaEnv

Privacy-aware attention navigation benchmark for resource constrained AI agents.
Motivated by the real-world constraints of on-device AI assistants like Meta's Ray-Ban Surfer.

## What is FoveaEnv?
FoveaEnv is a reinforcement learning environment for training privacy-aware attention navigation agents.

## Features
- Custom RL environment
- Gradio interface
- Real-time visualization
